# 🏥 ADHI — Landing Page

> Landing page institucional desenvolvida para a **Associação ADHI — Angra dos Reis**, voltada ao apoio de diabéticos, hipertensos e pessoas que buscam mais qualidade de vida.

## 📌 Sobre o Projeto

Este projeto é uma landing page freelancer desenvolvida para a ADHI (Associação de Diabéticos e Hipertensos de Angra dos Reis). O objetivo é apresentar os benefícios da associação, facilitar o acesso ao manual de cuidados e incentivar novos membros a ingressar na comunidade.

## ✨ Funcionalidades

- Seção hero com chamada principal e métricas da associação
- Botão de download do manual de cuidados
- Seção de benefícios para membros
- Seção de comunidade
- FAQ (Dúvidas frequentes)
- Botão de entrada no grupo via WhatsApp
- Design responsivo

## 🛠️ Tecnologias Utilizadas

- **React** + **Vite**
- **CSS / Tailwind CSS**
- **HTML5**

## 🚀 Como Rodar Localmente

```bash
# Clone o repositório
git clone https://github.com/LucasTesta444/adhi-landing-page.git

# Acesse a pasta
cd adhi-landing-page

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
```
